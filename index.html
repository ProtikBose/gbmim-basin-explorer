How to Use

Use the Reservoir / Barrier / Both tabs to switch data layers. Search by ID to find a specific site. Click any marker to load its watershed and downstream path.

Dataset Details

Reservoir locations from GRanD v1.3 supplemented with the GBMIM dataset covering Ganges–Brahmaputra–Meghna–Irrawaddy–Mekong systems. Watershed delineations use HydroSHEDS DEM at 15 arc-second resolution.
